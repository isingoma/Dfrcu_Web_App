### Getting Started and Installation

1. Clone the repository:
```bash
git clone https://github.com/isingoma/Dfrcu_Web_App.git
```

2. Install dependencies:
```bash
cd reat-payment-app
npm install
```

3. Create a `.env` file based on `.env.example`:
```bash
cp .env.example .env
```

4. Start the development server:
```bash
npm run dev
```
